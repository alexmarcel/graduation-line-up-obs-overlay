👋 Getting Started
1
Import CSV — click the CSV button in the sidebar. Format: ID, Name, Group, Marks
2
Load Photos — click Photos and select a folder. Files must be named by student ID (e.g. 5190.jpg)
3
Load Audio — optional. Click Audio and select a folder of .wav or .mp3 files named by student ID (e.g. 5190.wav). Use the Audio: ON/OFF button to enable or mute playback on scan
4
Open OBS Mode — in a second window, click OBS Mode. Add that window as a Browser Source in OBS. Photos and audio sync automatically via browser storage. If the OBS window is opened fresh, use the Photos and Audio buttons in the bottom-right corner of the OBS window to reload them
5
Scan or Search — scan a barcode with a USB scanner, or use Search / Manual to find a student
